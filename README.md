# 🎮 Simple Guessing Games Website

A fun and interactive web-based platform featuring multiple guessing games with progressive difficulty levels. Users must log in to play and track their progress across different challenges.

---

## 🚀 Features

* 🔐 User Authentication (Login required)
* 🎯 Multiple guessing games (Number & Puzzle)
* 📈 Level-based progression system
* 🧠 Increasing difficulty as you advance
* 💻 Simple and clean user interface

---

## 🕹️ Games Included

### 🔢 Number Guessing Game

* Guess the correct number within a limited number of attempts
* Difficulty increases with each level

### 🧩 Puzzle Guessing Game

* Solve puzzles to advance to the next level
* Challenges become more complex over time

---

## 🔑 Authentication

Users must log in before playing:

* Secure login system
* Tracks user progress and levels
